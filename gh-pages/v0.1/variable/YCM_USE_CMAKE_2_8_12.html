YCM_USE_CMAKE_2_8_12Â¶

Use CMake modules imported from CMake 2.8.12.

Some modules you would like to use might not available on CMake 2.8.11, or might have received some bugfix or new feature in CMake 2.8.12. If this variable is enabled, YCM will allow you to use the new version of these modules with CMake 2.8.11 or older versions.

This option is available only if CMAKE_VERSION is 2.8.11 or older.
